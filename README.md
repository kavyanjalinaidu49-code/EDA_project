# Exploratory Data Analysis (EDA) Project

## 📌 Project Title
Student Performance Data Analysis

---

## 📖 Project Description
This project performs Exploratory Data Analysis (EDA) on a student performance dataset using Python libraries such as Pandas, Matplotlib, and Seaborn.

The main objective of this project is to analyze student scores, identify patterns, detect missing values, and visualize the data using graphs.

---

## 🎯 Objectives
- Understand dataset structure
- Analyze student performance
- Find trends and patterns
- Detect missing values
- Create visualizations for better understanding

---

## 🛠 Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

---

## 📂 Dataset Used
Students Performance Dataset (CSV file)

---
